== README

Run <tt>$ruby run.rb</tt>.
You can change game setting in file game.rb (columns, rows and number of iterations).

You can place your file with coordinates to root folder with name <tt>inp.txt</tt>.
Format for coordinates could be fill in with comas and semicolons, where first digit is Y and second on is a place of alive cell on X that you want to put inside.
Example:

1,2; 1,3; 1,4; 1,5; 2,3;
